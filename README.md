# SMS Device Control using GSM Module and Arduino UNO

## 📌 Overview
This project allows users to control electrical devices using SMS commands via SIM900 GSM module.

## 🧰 Components
- Arduino UNO
- SIM900 GSM Module
- LED / Relay Module
- SIM Card
- Jumper wires

## 📱 SMS Commands
ON  -> Turns device ON
OFF -> Turns device OFF

## 🔌 Connections
SIM900 TX -> Arduino pin 8
SIM900 RX -> Arduino pin 7
LED -> Arduino pin 6
